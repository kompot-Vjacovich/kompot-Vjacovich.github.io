Значение block

  • Блочные элементы располагаются один над другим, вертикально.
  • Блок стремится расшириться на всюдоступную ширину. Можно указать ширину и высоту явно.
  • Такое значение свойства display некоторые элементы имеют по умолчанию: div, h1-h6, p

Значение inline

    Элементы располагаются на той же строке, последовательно. Ширина и высота элемента определяются посодержимому. Поменятьих нельзя. Инлайновые элементы по умолчанию: span, a, b
Ширина Игнорируется

Значение inline-block

  • Располагаются в строке друг за другом.
  • Размер устанавливается по содержимому.
  • Работают свойства width/height

Значение flex

  • flex-direction: направление элементов внутри flex-контейнера (column/row/column-reverse/row-reverse)
  • justify-content: выравнивание элементов по главному направлению (flex-start/flex-end/center/space-evenly/space-between/space-around)
  • align-items: выравнивание элементов по второстепенному направлению (flex-start/flex-end/center/stretch/baseline)
