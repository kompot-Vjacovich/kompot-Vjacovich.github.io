Основные HTML теги

Тег Предназначение тега
<html> </html> Указывает программе просмотра страниц, что внутри будет HTML документ
<head> </head> Определяет место, где помещается различная информация не отображаемая в теле документа. Здесь располагается тег названия документа и прочие теги для настройки страницы.
<body> </body> Определяет видимую часть документа
